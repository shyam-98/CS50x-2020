unix cli

File Commands

Display path of the current working directory

pwd

Make directory dir

mkdir dir

Change directory to dir

cd dir

Go to directory one level up

cd ..

List files

ls

Creat a file

touch filename

Delete a file

rm file

Process Management

Display currently active processes (static)

ps

Display all running processes (dynamic)

top

Kill process with id pid

kill pid

Searching

Search for pattern in files

grep pattern files

Search recursively for patter in dir

grep -r pattern dir

Search for pattern in the output of command

command | grep pattern

find all instances of file

locate file

System Info

Show current uptime

uptime

Find the username of current user

whoami

See kernel information

uname -a

File Permissions

Change permission of <file> to 775

chmod 775 <file>

Change permissions of <directory> (and its contents) to 600

chmod -R 600 <directory>

Change ownership of <file> to <user> and <group>

chown <user>:<group> <file>
